# Конвертер сборников песен из SoftProjector в Holyrics

Этот инструмент позволяет конвертировать сборники песен из формата `.sps` (SoftProjector) в формат, совместимый с Holyrics. Проект добавляет нумерацию к названиям файлов песен для более удобной организации.

## Что он делает?

*   **Конвертирует:** Преобразует экспортные файлы песенников SoftProjector (`.sps`) в отдельные текстовые файлы для каждой песни.
*   **Организует:** Каждая песня сохраняется в отдельный файл `.txt`, готовый для импорта в Holyrics.
*   **Нумерует:** В название каждого файла добавляется номер песни из сборника для лёгкой идентификации и сортировки.
*   **Работает локально:** Вся обработка происходит в вашем браузере. Никакие данные не отправляются на сервер, что обеспечивает полную конфиденциальность.

## Как использовать

1.  **Экспортируйте песни из SoftProjector:** В программе SoftProjector экспортируйте ваш сборник песен. В результате вы получите файл с расширением `.sps`.
2.  **Загрузите файл:** Откройте веб-страницу конвертера и загрузите ваш `.sps` файл, используя поле для выбора файла.
3.  **Скачайте архив:** После обработки будет предложено скачать `.zip` архив. Внутри архива вы найдете папки, соответствующие вашим сборникам, с текстовыми файлами песен.
4.  **Импортируйте в Holyrics:**
    *   Распакуйте скачанный архив.
    *   В программе Holyrics перейдите в меню *Песни* -> *Импортировать* -> *Другое*.
    *   Выберите опцию *TXT File (UTF-8)* и укажите путь к файлам песен.

## Для разработчиков

Этот проект представляет собой одностраничное веб-приложение, написанное на HTML и JavaScript.

### Основные технологии

*   **HTML5/JavaScript:** Основа приложения.
*   **sql.js:** Используется для чтения и обработки файлов `.sps`, которые являются базами данных SQLite.
*   **JSZip:** Применяется для создания `.zip` архива с конвертированными песнями прямо в браузере.

### Логика работы

Скрипт определяет тип загруженного файла (`.sps` на основе SQLite или старый текстовый формат) и соответствующим образом извлекает данные о песнях. Затем он форматирует текст каждой песни, добавляя специальные теги для заголовков (куплеты, припевы и т.д.), которые Holyrics может распознать. Наконец, все обработанные песни упаковываются в `.zip` архив.

Имена файлов генерируются по шаблону: `[Номер песни] — [Название песни].txt`.
